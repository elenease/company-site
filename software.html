Robert

Мобильное приложение для программно-аппаратного комплекса домашней автоматизации умного дома ROBERT.

Приложение разработано с нуля для группы компаний “Третий Трест” по представленному макету.

Состоит из более чем 10 экранов.

Адаптировано под версии Андроид 5.0 (API 22) и выше.

Имеет широкий спектр возможностей:

  • логин по биометрическим данным пользователя,
  • отправка push-уведомений с помощью FireBase,
  • использование web-sockets для обмена данными с базой данных,
  • серверная часть реализована на NodeJS,
  • подключение сторонних IP-видеокамер с кодеком H.264,
  • трансляция аудио-, видео-потоков с подключенных к приложению камер (WebRTC),
  • отображение полученных данных приборов учёта энергоресурсов в графической форме.

Панель оператора (Dashboard)

Второй проект для группы компаний "Третий трест". Частично использует функционал Панели администратора. Предназначен для мониторинга и управления событиями, оправленными оборудованием системы умного дома ROBERT. Реализован следующий функционал:
